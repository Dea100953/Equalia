# Equalia - Women's Empowerment Platform

## Welcome to Equalia 💜

Equalia is a dedicated, inclusive feminist website designed for every woman to feel safe, supported, and empowered. This platform provides a space where women can connect, share experiences, express emotions, stay informed about women's rights issues, and access emergency support.

---

## 🌟 Features

### 1. **Homepage**
- Beautiful welcome section with an inspiring introduction
- Quick navigation buttons to all main features
- Information about what Equalia offers
- Features section highlighting the platform's benefits

### 2. **Latest News Page**
- Current news about women's rights issues worldwide
- Featured stories from Afghanistan, Iran, Pakistan, and other countries
- News cards with images and descriptions
- Coverage of topics including:
  - Women's education rights
  - Personal freedoms and rights
  - Gender pay gap
  - Maternal health
  - Workplace harassment and accountability
  - Women in leadership

### 3. **Community Chatroom**
- Active, safe space for women to chat and express emotions
- Share experiences and support one another
- Community guidelines to ensure respectful dialogue
- Real-time messaging interface
- Simulated responses from community members

### 4. **Emergency Contacts**
- Country-specific emergency contacts and support services
- Multiple categories including:
  - Domestic violence support
  - Sexual assault support
  - Mental health crisis lines
  - Suicide prevention
  - Emergency services
- Coverage of 15+ countries
- Quick access to crisis helplines
- Information about each service

## 🚀 Getting Started

### Installation

1. **Clone or download this repository**
   ```bash
   git clone https://github.com/yourusername/Equalia.git
   cd Equalia
   ```

2. **Open in your browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # or
     npx http-server
     ```
   - Then navigate to `http://localhost:8000`

## 🆘 Emergency Support Information

Each emergency contact card includes:
- **Service Name**: What type of support they provide
- **Contact Number**: Phone number or alternative contact method
- **Description**: What each service offers
- **Link**: Website or more information (when available)

**Important**: If you're in immediate danger, always call your local emergency number first (911 in USA, 999 in UK, etc.).
